# Pharmacogenomics Card

> This report is generated by `/genome-report`. Run the skill after importing your genome data and completing onboarding.

## What This Will Contain
- Full enzyme status table (CYP2D6, CYP2C19, CYP2C9, CYP1A2, CYP3A4, UGT1A1, DPYD, TPMT, NAT2)
- Activity scores and metabolizer phenotypes
- Drug interaction matrix with CPIC/DPWG guideline references
- Carrier status flags (HFE, PNPLA3, etc.)
- Personalized dosing considerations
