# Setup Checklist

Track your progress through vault setup. Tick each box as you complete it. See [[Getting Started]] for detailed instructions on every step.

---

## Obsidian Setup
- [ ] Community plugins enabled (Settings > Community plugins > Restricted mode OFF)
- [ ] Dataview plugin installed and enabled
- [ ] Templater plugin installed (recommended)
- [ ] Periodic Notes plugin installed (optional, for biomarker tracking)

## Data Import
- [ ] Raw genome file placed in `data/raw/`
- [ ] `/genome-import` completed successfully
- [ ] QC report reviewed (SNP count, chromosome distribution)
- [ ] `data/genome.db` exists and is populated

## Onboarding
- [ ] `/genome-onboard` completed
- [ ] Health goals selected
- [ ] [[Dashboard]] shows personalized content
- [ ] `Reports/Wallet Card.md` generated — consider printing it
- [ ] Gene notes created (check Genes/ folder)
- [ ] [[Action Items]] populated

## Imputation (Optional)
- [ ] Read [[Imputation Guide]]
- [ ] Data exported for imputation server
- [ ] Imputed VCF files received
- [ ] `/genome-import` re-run with imputed data
- [ ] Gene notes updated with expanded data

## First Exploration
- [ ] Reviewed [[Dashboard]] overview
- [ ] Read your Wallet Card
- [ ] Opened at least one gene note from the top-priority list
- [ ] Browsed [[MoC - All Systems]] or [[Question Index]]
- [ ] Reviewed [[Action Items]] for recommended lab tests

## Ongoing Maintenance
- [ ] First lab results imported via `/biomarker`
- [ ] First vault audit via `/genome-audit`
- [ ] First research check via `/genome-analytics`

---
