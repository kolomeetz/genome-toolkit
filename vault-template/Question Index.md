# Question Index

Search by concern, not by gene. This is the most empathetic entry point into your genome.

## Health & Safety
- **Is this drug safe for me?** -> Start with [[Wallet Card]], then [[Pharmacogenomics Card]]
- **What should I tell my prescriber?** -> [[Prescriber Summary]]
- **What tests should I get?** -> [[Complete Testing Guide]]
- **Am I at risk for liver problems?** -> Check PNPLA3, HFE in [[MoC - All Genes]]
- **Should I worry about my heart?** -> Check APOE, LPA, PCSK9 in [[MoC - All Genes]]

## Understanding Myself
- **Why am I anxious?** -> Look for Anxiety profile in [[MoC - Phenotypes]]
- **Why do I react strongly to caffeine?** -> Check CYP1A2, ADORA2A
- **Why is sleep difficult?** -> Check CLOCK, PER2, ADORA2A
- **Why do I feel things intensely?** -> Check COMT, SLC6A4, FKBP5

## Behavioral Change
- **What exercise works best for my genes?** -> Check BDNF, PPARGC1A, ACTN3
- **How do I build habits?** -> Check dopamine system genes (DRD2, COMT, DAT1)
- **What supplements are evidence-based for me?** -> [[Interventions Dashboard]]

## Treatment Decisions
- **Is my SSRI working optimally?** -> Check CYP2D6, CYP2C19, CRP levels
- **Should I try a different medication?** -> [[Pharmacogenomics Card]]
- **What about supplements?** -> [[Interventions Dashboard]] tier 2-3

## For My Doctor
- **Emergency reference** -> [[Wallet Card]] (print this!)
- **Full briefing** -> [[Prescriber Summary]]
- **PGx details** -> [[Pharmacogenomics Card]]
